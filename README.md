# MNIST Digit Recognition using CNN

## Overview

This project implements a Convolutional Neural Network (CNN) using TensorFlow and Keras to classify handwritten digits from the MNIST dataset.

## Features

* Image preprocessing and normalization
* Convolutional Neural Network (CNN)
* Max Pooling layer
* Softmax classification
* Prediction visualization using Matplotlib

## Technologies Used

* Python
* TensorFlow
* Keras
* NumPy
* Matplotlib

## Dataset

MNIST Handwritten Digit Dataset

## Model Architecture

* Conv2D (32 filters, 3x3)
* MaxPooling2D
* Flatten
* Dense (10 classes, Softmax)

## Results

Achieved high classification accuracy on the MNIST test dataset.
